# Документация по запуску бота
1. Клонировать ветку main репозитория
2. Создать venv
3. Установить библиотеки из [requirements.txt](requirements.txt)
4. Создать базу данных и таблицы при помощи запросов из файла [/DB/create.sql](DB/create.sql)
5. Скопировать файл [DB/access-example.py](DB/access-example.py) в новый файл `DB/access,py` и заполнить его данными для подключения к созданной базе данных.
6. Создать сообщество ВК по инструкции [[group_settings.md]], cгенерировать токен, скопировать его и APP_ID в файл tokens.py, взяв за образец [[tokens-example.py]]
7. Заполнить файл токенами, и адресом сервера авторизации (redirect_uri)
8. При необходимости настроить туннель для сервера авторизации
9. Запустить веб-сервер авторизации auth_server.py
10. Запустить бота
