# Документация по использованию бота
1. На странице сообщества войдите в `сообщения сообществу`
2. Напишите любой текст
3. Бот попросит авторизоваться в ВК и выдать ему разрешения, чтобы он мог осуществлять для вас поиск по профилям
4. Если бот не смог определить ваш возраст он запросит диапазон возрастов для подбора профилей
5. Бот запросит пол людей которых вы хотите найти
6. Если бот не смог определить ваш город он запросит город для подбора профилей
7. Эти же параметры можно настроить произвольно в любой момент
8. Далее выводится запрос на поиск анкет
9. После поиска выводится первая анкета для поиска
10. Доступны действия по кнопкам: Дальше, В Избранное, В черный список.
11. Аналогично происходит навигация по избранному.
